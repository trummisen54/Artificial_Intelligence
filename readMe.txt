Make sure you have following files in your directory, in order to run the 'pioneer' example:


1. vrep.py

2. vrepConst.py

3. the appropriate remote API library: "remoteApi.dll" (Windows), "remoteApi.dylib" (Mac) or "remoteApi.so" (Linux). You should be able to find it under '$PATH_TO_V-REP3\V-REP_PRO_EDU\programming\remoteApiBindings\lib'

4. pioneer.py

5. env.py
